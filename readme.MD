#Rotation String (PHP)

Considere que você tem um método isSubString que verifica se uma palavra é
parte da outra (está contida em outra). Dado duas strings, s1 e s2, escreva
um código para verificar se s2 é uma rotação de s1 usando apenas uma chamada
para o método isSubString.
Ex.: "vistaremotaentre" é uma rotação de "entrevistaremota"
"entrevistaremota" é uma rotação de "trevistaremotaen"
